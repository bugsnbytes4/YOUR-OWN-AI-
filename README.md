# YOUR-OWN-AI-
A fully working vector database built from scratch in Python, with a web UI. Implements **HNSW**, **KD-Tree**, and **Brute Force** search side-by-side, plus a RAG pipeline powered by a local LLM via **Ollama**. This is a Python port of the original C++ project — same architecture, same algorithms, no C++ toolchain required.
